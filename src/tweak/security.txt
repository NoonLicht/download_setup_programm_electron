.Отключить защиту сети в Microsoft Defender Exploit Guard :Позволяет отключить защиту сети в Microsoft Defender Exploit Guard. После применения изменений, функциональность защиты сети будет отключена.

.Отключить в Microsoft Defender защиту в реальном времени :Отключает функцию обнаружения потенциально нежелательных приложений в Microsoft Defender. После применения изменений, программа не будет автоматически обнаруживать и блокировать ПНП (PUA).

.Отключить аудит событий, возникающих при создании или запуске процесса :Отключает аудит событий, связанных с созданием или запуском процессов в системе. После применения изменений, эти события не будут регистрироваться в журнале аудита.

.Не проверять приложения и файлы фильтром SmartScreen в Microsoft Defender :Отключает функцию проверки приложений и файлов фильтром SmartScreen в Microsoft Defender. После внесения изменений, система больше не будет использовать SmartScreen для проверки подлинности и безопасности приложений и файлов. 

.Отмечать файлы, скачанные из интернета как небезопасные :Включает функцию автоматического отмечания файлов, скачанных из интернета, как потенциально небезопасных. После применения изменений, файлы, загруженные из сети, будут помечаться как потенциально опасные, предупреждая пользователя о возможных рисках.

.Отключить Windows Script Host :Отключает Windows Script Host (WSH), что означает запрет выполнения скриптов и сценариев, использующих WSH на операционной системе Windows. WSH обеспечивает среду выполнения для различных скриптовых языков, таких как VBScript и JScript, и может использоваться для автоматизации задач и сценариев.

.Windows Sandbox :Windows Sandbox - это интегрированное в операционную систему Windows средство виртуализации, предназначенное для безопасного и временного запуска приложений или файлов в изолированной среде. Этот твик активирует и настраивает ''Windows Sandbox'' для использования.
